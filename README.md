# Candidatura Analista de Dados Senior (2025)

A **MUVA** procura um(a) Analista de Dados Senior para liderar a recolha, tratamento e análise de dados relacionados com a implementação das actividades nos projectos da **MUVA**.

Leia atentamente os **Termos de Referência (TDRs)** antes de submeter a sua candidatura. As informações recolhidas serão utilizadas exclusivamente para fins de recrutamento e tratadas com total confidencialidade.
